NEO QC UPDATED

v1.1.0

An update has been installed. Here are the key improvements in NeoQC v1.1.0:

  • 📋 Collapsible Activity Log Tray: Space-saving minimizable log tray in Ticket Edit Modal, contractible to reclaim workspace.
  • 🌡️ Real-Time Telemetry Sync: Real-time display slots showing Min, Max, and Avg CPU/GPU temperatures streamed from client machines.
  • 🔌 Real-Time Port Checker Matrix: Dedicated validation grid showing specific statuses (USB, Display, Audio, RGB) linked in real-time.
  • 📱 Redesigned Client Extension Mode: Brand new client layout designed as a physical extension of active tickets with manual port toggles.
  • 🌈 SignalRGB Installer & Fallback: Wrap SignalRGB installer in elevated PowerShell to bypass Windows UAC permission limits, and integrate OpenRGB CLI fallback sync.
