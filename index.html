    NEO QC UPDATED

    v1.4.2

    An update has been installed. Here are the key improvements in NeoQC v1.4.2:

    • Print / PDF Fixed: Report generation no longer fails with "pc is not defined" — printing and Save-PDF work again from the completed-builds table and the ticket modal.
    • Clean Component Names: Manufacturer part codes like "(100-100001277WOF)" are stripped everywhere you see components — dropdowns, ticket specs, the completed-builds table, customer views and the printed report — while meaningful details like "(White)", "(32Gbx1)" or "(2023)" are kept.
    • Update Check On Landing: Returning to the mode-selector screen now re-checks for updates automatically (rate-limited), so machines that stay open for days still pick up new versions without a restart.
