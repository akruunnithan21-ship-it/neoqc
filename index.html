NEO QC UPDATED

v1.0.1

An update has been installed successfully. Here are the key improvements in NeoQC version 1.0.1:

  • 🛠️ NSIS Installer Lockout: Locked out per-user installation (Developer Mode) by default in installer configurations.
  • 💾 Selected Ticket Persistence: Selected tickets on the Welcome screen now correctly persist into the Testing Client console.
  • 🔌 Detailed Ports Check: Integrated real PowerShell-based scanning of active USB devices, audio jacks, and video displays.
  • 🖥️ Live ARGB Sync Test Card: Built an interactive PC chassis representation that reflects accent colors, speed, and sync states in real time.
  • 📊 Telemetry HUD Alignment: Expanded layout width to 1180px and applied wrap-prevention rules to avoid text overlapping or squishing.
  • ⏱️ Cinebench Duration Limits: Enforced strict duration limits with a backend killer and a CPU-aware score estimation fallback to prevent system hanging.
  • ⚙️ Sidebar settings modal: Redesigned the settings interface into tabs (General, QC Thresholds, Advanced) for better organization.
  • 🔒 Deadline Lock: Deadline modification is now locked behind a password challenge (neoadmin) to prevent accidental resets.
  • 🧪 Checklist Consolidation: Consolidated Ports & Connectors checklist items under Physical Tests & Checks.
